Tic Tac Toe Game Project

Technologies: Java, OOPs Concepts

Developed a console-based Tic Tac Toe game using Java and Object-Oriented Programming concepts.
Implemented a User Board system to manage player moves, game status, and winner checking logic.
Used OOP concepts like Encapsulation, Abstraction, Inheritance, and Polymorphism for clean and modular design.
Created separate classes such as Main Class, Player Class, and Board Class to improve code organization and maintainability.
Added features like two-player gameplay, input validation, board display, and win/draw condition checking.
Designed the project with reusable methods and structured logic for better scalability and readability.


Train Ticket Booking System

Technologies: Java, OOPs Concepts

Developed a Train Ticket Booking System using Java with a Low-Level Design (LLD) approach and Object-Oriented Programming principles.
Designed classes such as Passenger, TicketBooking, Train, and Main Class for modular and structured implementation.
Implemented functionalities like ticket booking, seat allocation, passenger details management, and ticket cancellation.
Applied OOP concepts including Encapsulation, Abstraction, Inheritance, and Polymorphism to improve code maintainability and scalability.
Added validation for seat availability and booking operations with efficient data handling logic.
Followed clean coding practices with proper class separation, reusable methods, and user-friendly console interaction.
